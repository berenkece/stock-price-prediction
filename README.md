# AAPL Stock Direction Prediction

Predicts whether Apple stock will go up or down the next day using XGBoost.

## Features
- RSI (Relative Strength Index)
- Moving Average difference
- Daily Return
- Volatility

## Results
- Accuracy: ~55%
- Model: XGBoost Classifier

## Setup
```bash
pip install -r requirements.txt
python3 stockprediction_1.py
```
